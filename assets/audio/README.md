# Assets — Audio

| Cartella | Contenuto |
|---|---|
| `sfx/` | Effetti sonori (sparo, salto, danno, morte, boss, game over) |
| `music/` | Musica di background per ogni livello e menu |

## Sorgenti consigliate

- [Freesound.org](https://freesound.org) — SFX liberi
- [OpenGameArt.org](https://opengameart.org) — Musica chiptune gratuita
- [itch.io Audio](https://itch.io/game-assets/free/tag-music) — Pack completi

## Formati

Usare `.ogg` per la musica e `.wav` per gli SFX (Godot li gestisce entrambi nativamente).
