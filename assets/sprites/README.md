# Assets — Sprites

Inserire qui tutti gli sprite del gioco, organizzati per sottocartella:

| Cartella | Contenuto |
|---|---|
| `player/` | Spritesheet del personaggio (idle, run, jump, fall, die) |
| `enemies/` | Spritesheet per ogni tipo di nemico |
| `boss/` | Spritesheet del boss (fase 1 e fase 2) |
| `projectiles/` | Proiettili player e nemici |
| `tiles/` | Tileset per il livello (ground, platforms, background) |
| `ui/` | Icone HP, logo, font bitmap, bottoni |
